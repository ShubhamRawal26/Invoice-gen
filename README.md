# InvoiceGen – Simple Invoice Generator

A lightweight personal web tool to create professional invoices and export them as PDF.

## Live Demo
https://shubhamrawal26.github.io/Invoice-gen/

## Features
- Create invoices in-browser
- Auto calculations (subtotal, tax, total)
- Branding (color & logo)
- Save invoice as PDF
- Auto-save in browser

## Tech Stack
- HTML, CSS, JavaScript
